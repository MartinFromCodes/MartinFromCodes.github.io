1 准备工作，基本工具的安装

1.1 安装 python 2.7

python2.7 下载地址

我的电脑-属性-高级系统设置-高级-环境变量.找到path变量然后把python安装目录添加上去。

1.2 配置pip

将C:\Python27\Scripts 目录添加到系统环境变量

就安装好了wheel 工具可以安装 whl插件了

注意事项

关于中文乱码

  • 文件头加上 #coding=utf8 (必须)

  • 连接的时候加入charset设置信息
    pymssql.connect(host=self.host,user=self.user,password=self.pwd,database=self.db,charset=”utf8”)

  • sql语句中有中文的时候进行encode
    insertSql = “insert into WeiBo([UserId],[WeiBoContent],[PublishDate]) values(1,’测试’,’2012/2/1’)”.encode(“utf8”)
